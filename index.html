InstaPic

  • Used Ruby on Rails for the backend, and PostgreSQL to keeps track of the backend data.
  • Utilized React.js and Redux cycles for rendering different components on the frontend.
  • Adopted Active Storage(AWS) to store the posts in my seed file and the pictures users uploaded, which assures the safety of the posts.
  • Implemented Bcrypt and SecureRandom for backend data security along with session and current user state to provide front end authentication.
  • Utilized protected routes and switch/routes in ReactRouterDom, which restrict logged in/out users to certain pages.
